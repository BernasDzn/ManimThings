## dependencies

- manimgl
- fim

## installing dependencies

```python
# Install manimgl
pip install manimgl

# Clone repo
git clone https://github.com/3b1b/manim

# Get to project folder
cd manim

# Install dependencies
pip install -e .
```

## usage

### open cmd
- ```cd REPO_DIR```
- ```manimgl scene.py ClassName```
#### or
- ```manim-render -o filename scene.py ClassName```
